# Практична Робота №1: Базова Архітектура

### Стек Технологій

Бекенд: Node.js, Express (створення API)

База Даних: SQLite (файлова, безсерверна БД: server/restaurant.db)

Клієнт: Vanilla HTML, CSS, JavaScript (Fetch API)

---

## Інструкція для Запуску

### Ініціалізація Бази Даних - node server/init_db.js

### Запуск Сервера - node server/server.js

### Запуск Клієнта - index.html (open live server)
